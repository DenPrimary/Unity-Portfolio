# 2D Pong in Unity 6000.3.2f1

Classic Pong game built from scratch in Unity.

## Features
- Ball physics with speed curve (ball accelerates after each hit)
- AI opponent (follows the ball)
- Score system (first to 5 points wins)
- UI: score display, win message, restart button

## How to play
- Player: move paddle with **Up/Down** arrows
- **Esc** is for exit
- AI controls the right paddle

## What I learned
- Collision detection in Unity
- UI management via C#
- Simple AI logic
- Game state management (gameplay, scores, win\loss)
